Changelog:
Biome Spawn Point
https://curseforge.com/minecraft/mc-mods/biome-spawn-point

=========
v2.1: (1.20.*) Added official support for NeoForge. Just like Fabric and Forge, in the same jar!
Updated structure and content of mod info files.

=========
v2.0: A big backend update!
Forge and Fabric (and Quilt) are now contained in one file.

This also means that the two separate project pages for Forge/Fabric will be converted to one (on CurseForge). While having a clean file list! The Fabric project page on CF will remain as a legacy for older Fabric versions.

=========
v1.7: (Fabric) Updated mod to Minecraft version 1.19.3.

=========
v1.6: Fixed Biome Spawn Point and Village Spawn Point compatibility.
The mod now works correctly server-side with the Collective 5.17+.
Now logs messages correctly when creating a new world.

=========
v1.5: Fixes the problems caused by Mojang's launcher update.

=========
v1.4: Added Mod Menu support.

=========
v1.1-1.3: Added support for Village Spawn Point. With both installed and configured, you now spawn in a village near the chosen biome.

v1.0: Initial release of Biome Spawn Point.
