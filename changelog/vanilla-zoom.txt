Changelog:
Vanilla Zoom
https://curseforge.com/minecraft/mc-mods/vanilla-zoom

=========
v3.0: (1.21.9 Forge) Fixed incorrect remnant files included in the mod jar.

=========
v2.9: Updated mod to use Collective's new RegisterKeyMappingsHelper.

=========
v2.8: (Forge & Neo) Removed redundant event register annotations.
- This fixes events loading when they shouldn't.

=========
v2.7: Added extra null check for the zoom hotkey.

=========
v2.6: Added Bundle support. The mod can now be enabled/disabled via bundle configs.
(NeoForge) Fixed the in-game config screen not working correctly.

=========
v2.5: (1.20.5 Neo) Updated to work with NeoForge's TickEvent changes in 20.5.14.

=========
v2.4: (1.20.5) Fixed the pack.mcmeta file having an incorrect structure.

=========
v2.3: Improved mod icon compression, which lowers the jar file size.
Updated mod URLs to the new GitHub location.
(NeoForge) Updated to the new changes: Added 'type=' to mods.toml, and moved the modEventBus field.

=========
v2.2: (1.20.*) Added official support for NeoForge. Just like Fabric and Forge, in the same jar!
Updated structure and content of mod info files.

=========
v2.1: Fixed the zoom not working as intended when using custom keybinds.

=========
v2.0: A big backend update!
Forge and Fabric (and Quilt) are now contained in one file.

This also means that the two separate project pages for Forge/Fabric will be converted to one (on CurseForge). While having a clean file list! The Fabric project page on CF will remain as a legacy for older Fabric versions.

=========
v1.2: (Fabric) Updated mod to Minecraft version 1.19.3.

=========
v1.1: Added Mod Menu support.

=========
v1.0: Initial release of Vanilla Zoom.
