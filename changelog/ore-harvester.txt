Changelog:
Ore Harvester
https://curseforge.com/minecraft/mc-mods/ore-harvester

=========
v1.5: (Forge & Neo) Removed redundant event register annotations.
- This fixes events loading when they shouldn't.

=========
v1.4: Added Bundle support. The mod can now be enabled/disabled via bundle configs.
(NeoForge) Fixed the in-game config screen not working correctly.

=========
v1.3: Improved the spawn location of ore drops and stopping their velocity.

=========
v1.2: Improved compatibility with modded ores by enabling fuzzy search to determine what's an ore block.
Can be configured via the 'enableFuzzyOreSearch' config option if need be.

=========
v1.1: Fixed the mixin version being set incorrectly.

=========
v1.0: Initial release of Ore Harvester.
